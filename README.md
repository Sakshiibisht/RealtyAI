# RealtyAI - Smart Real Estate Insight Platform 
RealtyAI is an AI-powered real estate analytics platform that predicts housing prices and provides market insights using machine learning.

## Features
- Property price prediction using Random Forest
- Budget-based location recommendations
- Interactive market analysis dashboards
- Real estate dataset exploration
- Fully deployed web application

## Technologies Used
Python
Streamlit
Sikit-learn
Pandas
NumPy
Plotly
